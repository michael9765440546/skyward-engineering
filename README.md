skyward-engineering 🚀:

Just a collection of aerospace tools and simulators I'm building while learning the ropes of aeronautics.

Most of these are Python-based scripts to help visualize data or run quick calculations for things like lift and glide performance.

What's in here?
Airfoil Lift Simulator:A Python tool for calculating lift based on wing parameters and speed.

GlideSafe: Working on glide ratios and safety margins. I’ve included some distance graphs in there to show the slopes.

Takeoff Distance Simulator: Quick way to check how much runway you'd actually need under different conditions.

Running the scripts:
If you want to try these out, you’ll probably need matplotlib and numpy installed.

Bash
pip install matplotlib numpy
Then just navigate to the folder and run the python file:

Bash
python "Airfoil Lift Simulator/airfoil_lift_simulator.py"
